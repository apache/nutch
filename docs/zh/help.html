搜索帮助

查询

 使用Nutch搜索, 只需键入一些关键词语.
  • 搜索结果只包括含有 所有查询词的网页.
  • 使用双引号将相邻的词作为短语包含起来, 例如, "New Zealand".
  • 英文单词之间的符号会触发短语匹配. 因此搜索http://www.apache.org/ 等同于搜索"http www apache org".
  • 搜索英文单词不区分大小写, 因此搜索NuTcH 等同于搜索 nUtCh.
  • 你可以在一个词前面加减号丛而禁止它出现在搜索结果中, 例如, 搜索football -nfl 会找到讨论football, 但不出现"nfl"的网页.

结果

 在搜索结果中每个匹配的页面都含有如下链结:
  • (网页快照) 显示Nutch下载该网页时的内容.
  • (评分详解)显示Nutch如何给该网页打分.
  • (anchors)显示指向该网页而被Nutch索引的anchor文本.
