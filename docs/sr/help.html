Упити

 Да бисте претраживали помоћу Nutch-а, само укуцајте неколико речи.
  • Резултати представљају само оне странице које садрже све речи из упита.
  • Користите наводнике око речи које се морају појављивати заједно, као фраза, нпр., "Нови сад".
  • Интерпункција између речи такође изазива претраживање по фразама. Тако да је претраживање по http://www.apache.org/ идентично као претраживање по "http www apache org".
  • Претраживање не разликује велика и мала слова у упиту, тако да је претраживање по NuTcH идентично као претраживање по nUtCh.
  • Да бисте искључили из резултата странице које садрже неки термин ставите знак минус испред тог термина, нпр., претраживање по фудбал -нфл ће пронаћи странице које за тему имају фудбал, али не користе реч "нфл".
  • То је то!

Резултати

 Свака страница у резултатима има следеће линкове:
  • (кеширано) приказује верзију странице коју је Nutch дохватио.
  • (објасни) приказује објашњење како је ова страница бодована.
  • (везе) приказује списак одредница коришћених да се направи линк ка овој страни.
