FAQ

Warum braucht die Welt Nutch, wo doch sowieso alle Suchmaschinen gratis sind?

Suchmaschinen sind in ihrer Benutzung gratis - genauso wie Fernsehen gratis ist. Aber genauso wie das Fernsehen können Suchresultate von den Institutionen manipuliert werden, welche sie kontrollieren. Die einzige Möglichkeit, um unvoreingenommene Suchresultate zu erhalten, besteht darin, das dazu nötige Ranking-System öffentlich zu machen. Das Ziel von Nutch ist es, eine qualitativ hochwertige Such-Technologie frei verf¨gbar zu machen.

Wie kann ich helfen?

Wenn Sie darin interessiert sind, uns zu sponsern, besuchen Sie die Spenden-Seite.

Wenn Sie an der Entwicklung interessiert sind, besuchen Sie die Entwickler-Seite.

Haben Sie andere Vorschläge, Anregungen oder Fragen, senden Sie doch eine Mail an general@nutch.org.

Wie kann sich eine gemeinnützige Organisation den Betrieb einer Suchmaschine leisten?

Nutch ist primär ein Software-Projekt und keine Dienstleistung. Der Einsatz im grossen Umfang wird vermutlich von kommerziellen Interessen, unabhängig von Nutch, ausgehen, finanziert durch Werbung oder ähnliches. Wenn die Software gut genug ist, werden vielleicht bereits existierende grössere Suchmaschinen Nutch an Stelle der vorhandenen closed-source Software verwenden.

Das Nutch-Projekt selbst wird möglicherweise ein kleines Demo-System hosten, um zu zeigen, dass das Ganze wirklich funktioniert.

Wird Nutch jemals so gut sein wie andere, kommerzielle Suchmaschinen?

Wir hoffen, es wird besser. Mit Entwicklern und Forschern rund um den Globus hoffen wir die Qualität zu übertreffen, welche eine einzelne Firma erreichen kann.

Wie kann ich Nutch davon abhalten, meine Seite zu indexieren?

Bitte besuchen Sie unsere Webmaster-Informations-Seite.

Wie kann ich sichergehen, dass Nutch meine Seite indexiert?

Nutch nutzt das DMOZ Open Directory als Ausgangspunkt für die Indexierung. Der beste Weg, Ihre Seite in den Nutch Index zu bekommen, ist also, sie in das Open Directory einzutragen.

Wird Nutch eine verteilte, P2P-basierte Suchmaschine?

Wir glauben nicht, dass es zur Zeit möglich ist, eine peer-to-peer Suchmaschine aufzubauen, welche mit bestehenden Suchmaschinen konkurrieren könnte. Sie wäre einfach zu langsam. Resultate in weniger als einer Sekunde auf dem Bildschirm zu haben, ist wichtig: so können Benutzer schnell Ihre Suchanfrage umformulieren, bis sie gefunden haben, was sie suchen. Kurzgesagt: Eine schnellere Suchmaschine ist die bessere Suchmaschine.

Trotzdem können Interessierte, wenn sie dies wollen, ein Unterprojekt starten, welches die Entwicklung einer verteilten Suchfunktionalität zum Ziel hat.

Nutzt Nutch einen verteilten Crawler wie Grub?

Verteiltes Indexieren kann Bandbreite sparen, aber langfristig ist diese Bandbreite nicht von Bedeutung. Eine erfolgreiche Suchmaschine braucht mehr Bandbreite, um Seiten mit Suchresultaten an den Benutzer zu schicken, als der Crawler Download-Bandbreite zum Sammeln von Seiten benötigt. Eine Reduktion der Indexierungsbandbreite würde also nur wenig zur Reduktion des gesammten Traffics beitragen.

Wird die öffentliche Verfügbarkeit nicht Manipulationen am Ranking-System erleichtern?

Suchmaschinen arbeiten hart an Ranking-Systemen, die eine Manipulation erschweren. Trotzdem schaffen es Such-Optimierer immer wieder, die Algorithmen zu knacken und so das Ranking ihrer Seiten zu verbessern. Viele Seiten nutzen zum Beispiel Link-Farmen, um Link-basierte Ranking-Systeme (wie z.B. das von Google) zu umgehen. Suchmaschinen-Betreiber vergelten dies, indem ihre Link-basierten Algorithmen verbessert werden, um den Effekt der Link-Farmen zu neutralisieren.

Mit einer Open-Source-Suchmaschine wird das nicht anders sein - einfach öffentlich. Dies verhält sich wie bei Verschlüsselungs- und Virus-Scanalgorithmen. Auf lange Zeit gesehen macht das Offenlegen solcher Algorithmen diese stärker, da mehr Leute den Sourcecode durchstöbern und damit Fehler finden und Verbesserungen anbringen können.

Wann wird Nutch Bilder und PDF-Dateien durchsuchen können?

Wir hoffen bald.
