JUnit Testpackage for Microsoft © PowerPoint document parsing plugin.

The example ppt-files are located in the subdirectory src/plugins/parse-mspowerpoint/samples. They are copied by the ant-task to the directory src/plugins/parse-mspowerpoint/data for testing. Additionaly to the ppt-files, there could also be stored files with the same name with postfix ".content" or ".title". If they exists, the test classes are checking the extracted content against these files.

Example

  • PowerPoint file to test: test.ppt
  • Reference file with content: test.ppt.content
  • Reference file with title and properties: test.ppt.title
