Agents

If you use Nutch to perform extensive crawls of sites that you do not control, please subscribe to the Nutch agent mailing list. This is where we encourage webmasters to post questions about the Nutch crawler.

The Nutch agent mailing list is : nutch-agent@lucene.apache.org.
